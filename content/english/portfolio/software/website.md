## Problem

I wanted a clean interface to demonstrate my past work, and I wanted to maintain control of the site.

## Solution

In typical Caleb fashion I gravitated to the solution that's cheap, opensource, easy to selfhost, and probably more complicated that it ought to be!  I've ""built"" a website, if setting up a Hugo website can be considered building. Have a look around and see how it's going. I've drawn heavily on [this writeup](https://schnerring.net/blog/create-a-hugo-website-with-github-pages-github-actions-and-cloudflare/) to get it up and running.  At the time of writing the site is hosted in GitHub Pages and media is hosted in pCloud.
